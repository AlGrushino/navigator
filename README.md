Реализованы обход графа в глубину(DFS) и ширину(BFS), к качестве контейнеров использовались стек и очередь, реализованные самостоятельно. Реализован алгоритм Дейкстры для поиска кратчайшего пути, алгоритм Флойда-Уоршелла для построения матрицы кратчайших путей между всеми парами вершин, алгоритм Прима для построения матрицы смежности минимального остовного дерева, а также муравьиный алгоритм для решения задачи коммивояжёра. Покрыл функции документацией и тестами.

How to run:

git clone

cd navigator/src

make

cd bin

./graphcli

Load graph from file

Choose algorithm


How to test:

git clone

cd navigator/src

make test
